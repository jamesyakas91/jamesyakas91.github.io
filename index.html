Software Development Portfolio

Welcome to my portfolio! Here you can find a selection of web applications I've developed.

Although aesthetics is not my forte, I am more concerned with functionality. I do try my best to create visually pleasing designs, but my primary focus is on building functional and efficient software solutions.

Happy Local

Happy Local is a website where users can search for food or drink specials based on their suburb.

What can I make?

What Can I Make? is a website where users can search for recipes based on their needs. The point in this project is to test my skills rather than anything else.

Chat Class

Chat Class is a website where users can create chats for their classes to discuss the topics, share and get help.

Stock Analyser

Stock Analyser is a website to find what day on average has the cheapest price for a given stock. It also provides analysis on the average days passed for a given price.

Gist Downloader

Gist Downloader is an application to download your Gists with varying options.

Insurance Calculator

Insurance Calculator calculates whether you are better off buying insurance or investing given parameters.

Cash Flow Calculator

Cash Flow Calculator calculates your cash flow for a given period of time.

Laptop Extractor

Laptop Extractor extracts specifications from laptop models from PB Tech website.
